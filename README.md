# SOMB
SOMB (Sofa Obj to MDD for Blender) est un programme qui permet de traiter des obj produits par le logiciel SOFA lors d'une simulation pour en faire un fichier MDD qui peut Ãªtre traiter par Blender

## Comment utiliser le SOMB:

Il suffit d'executer la commande ./somb dossier 
On lit alors les fichiers .obj que l'on transforme en un fichier res.mdd utilisable dans blender.
